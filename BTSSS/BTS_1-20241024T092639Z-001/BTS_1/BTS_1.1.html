Domanda 1

Bootstrap è un framework front-end gratuito per uno sviluppo web più rapido e semplice
Bootstrap include modelli di progettazione basati su HTML e CSS per tipografia, moduli, pulsanti,
tabelle, navigazione, modali, caroselli di immagini e molti altri, oltre a plugin JavaScript opzionali

Domanda 2

Per creare una pagina web utilizzando Bootstrap, devi importare il framework Bootstrap, che include principalmente
due componenti: i file CSS e i file JavaScript (JS).
Devi importare il file CSS di Bootstrap per utilizzare gli stili e le classi predefinite.
Puoi farlo collegando il CDN ufficiale di Bootstrap nel tuo <head>.

Ecco come includere i file tramite CDN:
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Domanda 3

Un CDN (Content Delivery Network) è una rete distribuita di server posizionati in varie parti del mondo,
progettata per distribuire contenuti web (come file JavaScript, CSS, immagini e video) in modo rapido ed efficiente.
I vantaggi principali di usare un CDN rispetto a scaricare Bootstrap localmente sul tuo sito:

Maggiore velocità di caricamento:

Il contenuto viene distribuito dai server più vicini all'utente. Questo riduce i tempi di latenza e accelera il caricamento della pagina.
Le reti CDN sono ottimizzate per gestire rapidamente grandi quantità di richieste in contemporanea.

Caching condiviso tra siti:

Se l'utente ha già visitato un altro sito che utilizza lo stesso file Bootstrap dallo stesso CDN, il file potrebbe già essere nella cache del browser.
Ciò significa che non dovrà essere scaricato di nuovo, accelerando ulteriormente il caricamento della tua pagina.

Domanda 5

La differenza tra .container e .container-fluid in Bootstrap è:

.container: ha una larghezza fissa che si adatta ai breakpoint di Bootstrap (piccolo, medio, grande schermo) e rimane centrato con margini laterali.

.container-fluid: è un contenitore che occupa sempre il 100% della larghezza della finestra, indipendentemente dalla dimensione dello schermo.
